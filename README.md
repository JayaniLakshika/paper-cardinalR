# cardinalR: Generating Interesting High-Dimensional Data Structures

This repo contains the materials to fully reproduce the paper titled "cardinalR: Generating Interesting High-Dimensional Data Structures".

## Description

- [`data`](data): The data sets used in the empirical application.
- [`figures`](figures): Figures used in the paper.
- [`scripts`](scripts): R scripts to reproduce the results presented in the paper.
- The manuscript is written using [Rmarkdown](https://rmarkdown.rstudio.com/). The TeX and PDF files for the paper are generated by rendering the [`paper-cardinalR.rmd`](paper-cardinalR.rmd) file.
